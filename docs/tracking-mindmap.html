Alcovia Tracking Architecture

Complete end-to-end tracking system for ad campaigns, organic traffic, and lead analytics

1. How a Lead is Tracked (End to End)

Paid Ad Flow:

Instagram Ad
->
User clicks ad
->
alcovia.life/lp/1?ad_id=...&utm_content=m1_fear
->
Meta Pixel fires PageView
->
User clicks CTA button
->
Registration form opens
->
Form submitted
->
Meta Pixel fires Lead + CAPI fires Lead
->
n8n webhook receives ALL data
->
ClickUp task + Google Sheet + Email + WhatsApp

Organic Flow:

User finds site via Google/WhatsApp/ChatGPT/Direct
->
Lands on any page
->
Browser records referrer
->
User clicks any CTA
->
Form opens (button tracked)
->
Form submitted with: traffic_source + referrer + landing_page + button

3. Data Captured Per Lead (19 Fields)

Form Data (9 fields)

student_name parent_name phone school grade person_type city email whatsapp_optin

Ad Attribution (5 fields)

ad_id which specific ad (e.g. alc_m1_fear_240326)
utm_source = meta
utm_medium = paid
utm_campaign = lp_leadgen_mar26
utm_content = m1_fear (messaging + tone)

Source Tracking (3 fields)

traffic_source = google / whatsapp / instagram / direct / chatgpt / etc.
referrer_raw = full referrer URL
landing_page = /lp/1, /, /meet-the-team, etc.

Behavior Tracking (2 fields)

form_open_source = which button opened the form (hero_cta, lp_navbar, lp_mobile_floating, navbar_desktop, etc.)
timestamp = when submitted

5. Ad Campaign Structure (25 Ads)

5 Messaging Categories: Real-world readiness, Career counselling, Time management, Peer group, Top 1%
5 Tones per Messaging: Fear, Aspirational, Authority, Myth Busting, Comparison
5 Landing Pages: /lp/1 (Fear), /lp/2 (Aspirational), /lp/3 (Authority), /lp/4 (Myth Busting), /lp/5 (Comparison)
Landing page is determined by tone, not messaging. All fear-toned ads go to /lp/1, all aspirational to /lp/2, etc.
Each ad has a unique ad_id: e.g. alc_m1_fear_240326 = Messaging 1, Fear tone, dated March 26.
MetricWhere to See
